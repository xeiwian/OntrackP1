1)  Begin with running the following command 'java -D"java.library.path=./DynamoDBLocal_lib" -jar DynamoDBLocal.jar -sharedDb' 
    in one terminal to start up the local database server.

2)  In another terminal, run the 'yarn start' command line to begin.

3)  Go to http://localhost:3000/ and it will display a webpage with a redeem coupon button. 

4)  Select the button to see the response.
